## How to Use

**Pick a resource, copy it, use it.** Each file is self-contained.

1. Browse a category above (or [visit the website](https://mnemoverse.github.io/vibe-coding-companion-pack/))
2. Open the file you need
3. Copy and use — replace `[PLACEHOLDERS]` with your details
4. Each prompt has a "tired version" for low-energy days

**With the book:** Each resource references its source appendix. Read the appendix for the "why", use the companion file for the "how".

**Without the book:** Checklists and templates work standalone. Prompts and weekend plans are richer with the book's context.
